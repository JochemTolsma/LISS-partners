LISS-Partners

Jochem Tolsma - Radboud University, the Netherlands

Last compiled on februari, 2021

This website is a replication package for the paper “LISS-partners” by Jeroense, Spierings, and Tolsma (n.d.). It contains R code to replicate all Tables in the manuscript.

Use the top menu to navigate to the section of interest. The section replicates the Main Tables of the manuscript.

To copy the code click the button in the upper right corner of the code-chunks.

Data

All data and scripts can be found on Github
